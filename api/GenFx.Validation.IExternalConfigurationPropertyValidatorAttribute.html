Interface IExternalConfigurationPropertyValidatorAttribute

Represents an System.Attribute that describes how a target component configuration property should be validated when set.

Namespace:GenFx.Validation
Assembly:GenFx.dll
Syntax
public interface IExternalConfigurationPropertyValidatorAttribute
Remarks

Attributes that implement this interface can be attached to components that need to describe how validation should be done for a configuration property external to the component when the component is being used. For example, a FitnessEvaluator type may require that a binary string entity have a specific length.
Note to developers: if creating your own PropertyValidatorAttribute, it is a best practice to also create a version of the attribute that implements the IExternalConfigurationPropertyValidatorAttribute interface. This allows third-party developers to use your attribute like they would a built-in GenFx attribute.
