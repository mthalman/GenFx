Class MultiDemeGeneticAlgorithm

A type of genetic algorithm that maintains multiple isolated populations and then migrates a select number of genetic entities between the populations after each generation.

Remarks

The number of GeneticEntity objects that migrate each generation is determined by the MigrantCount property value. Those GeneticEntity objects with the highest fitness value are the ones chosen to be migrated.
