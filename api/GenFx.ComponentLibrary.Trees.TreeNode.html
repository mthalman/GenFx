Remarks

The TreeNode objects of a tree contain the data making up the representation of that tree. For example, the data in the nodes of an expression tree represent the functions and terminators of the expression language.

Remarks

Notes to implementers: When this method is overriden, it is suggested that the CopyTo(TreeNode, TreeEntityBase, TreeNode) method is also overriden.
In that case, the suggested implementation of this method is the following:

public override object Clone(TreeEntity newTree, TreeNode newParentNode)
{
    MyTreeNode node = new MyTreeNode();
    this.CopyTo(node, newTree, newParentNode);
    return node;
}
