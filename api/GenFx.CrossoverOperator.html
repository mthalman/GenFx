Syntax
[DataContract]
public abstract class CrossoverOperator : GeneticComponentWithAlgorithm, INotifyPropertyChanged
Remarks

A CrossoverOperator acts upon two GeneticEntity objects that were selected by the SelectionOperator. It exchanges subparts of the two GeneticEntity objects to produce one or more new GeneticEntity objects. It is intended to be similar to biological recombination between two chromosomes.
