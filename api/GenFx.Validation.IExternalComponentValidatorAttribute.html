Interface IExternalComponentValidatorAttribute

Represents an System.Attribute that describes how a target GeneticComponent should be validated.

Namespace:GenFx.Validation
Assembly:GenFx.dll
Syntax
public interface IExternalComponentValidatorAttribute
Remarks

Attributes that implement this interface can be attached to components that need to describe how validation should be done for some other component when the attributed component is being used. Note to developers: if creating your own ComponentValidatorAttribute, it is a best practice to also create a version of the attribute that implements the IExternalComponentValidatorAttribute interface. This allows third-party developers to use your attribute like they would a built-in GenFx attribute.
