Class RankSelectionOperator

Provides a selection technique whereby the GeneticEntity objects in a Population are selected according to their fitness rank in comparison to the result of the Population.

Remarks

Selection is then based on this ranking rather than on absolute fitness. This technique avoids selecting only a few of highly fit GeneticEntity objects and thus can prevent premature convergence. But it also loses the perhaps important distinguishment of absolute fitness values later in a run. Use of a FitnessScalingStrategy object does not have an impact when RankSelectionOperator is being used since absolute differences in fitness are ignored.
