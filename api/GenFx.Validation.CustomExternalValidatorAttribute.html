Syntax
[AttributeUsage(AttributeTargets.Class, AllowMultiple = true)]
public sealed class CustomExternalValidatorAttribute : CustomPropertyValidatorBaseAttribute, _Attribute, IExternalConfigurationPropertyValidatorAttribute
Remarks

This class can be used when a custom PropertyValidator type has been defined but no arguments need to be passed to it. Rather than creating a new PropertyValidatorAttribute type, this class can be used instead.
