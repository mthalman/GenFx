Syntax
[DataContract]
public abstract class GeneticEntity : GeneticComponentWithAlgorithm, INotifyPropertyChanged
Remarks

A genetic entity in a genetic algorithm represents the "organism" which undergoes evolution. All the genetic operators such as the SelectionOperator, CrossoverOperator, and MutationOperator act upon genetic entities to bring about change in the system.
