[DataContract]
public abstract class ListEntityBase : GeneticEntity, INotifyPropertyChanged, IList, ICollection, IEnumerable
Remarks

This class does not define the list structure. It only defines the API. Classes which derive from ListEntityBase are responsible for the definition of the actual data structure representing the list.
